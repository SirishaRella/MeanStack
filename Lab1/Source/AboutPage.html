Knowledge Graph Search and Text to Speech API



Knowledge graph API:


The knowledge Graph Search API lets us find the entities in the Google Knowledge Graph. The current API we used retrieves the image of the person, name and description of the image.


Text to Speech API:


This API converts the text to speech format. In our project this API is used to tell about the person description that we retrieved in Knowledge graph API.
